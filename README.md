# 炸金花单机H5手机版
用v0.dev ai写的炸金花游戏（也要自己修改一些代码），完全单机的vue实现，没有ai玩家，全靠自己手动控制。可用于多个朋友线下一起玩，各自轮流操作即可。

## 环境要求
```shell
$ node --version
v20.15.1

$ npm --version
10.7.0
```
## 实现功能
* 发牌、看牌、比牌、下注、设置
* 各种声效和背景音乐设置
* 完全随机牌，没有任何后门
* 不需要服务器，可以部署到静态网站，或本地运行即可

## 安装运行
```
npm install
npm run dev
```
